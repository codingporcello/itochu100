# 伊藤忠100股冒險計畫 Claude版本

把長期存股遊戲化的個人 RPG 投資養成系統，採用 RO 仙境傳說風格 UI。
不是看盤軟體，是你的 RPG 角色頁面。⚔️

## 使用方式

直接用瀏覽器開啟 `index.html` 即可，無需安裝任何東西。
所有資料（冒險日誌、規則、設定）都儲存在瀏覽器的 localStorage。

也可以啟動本機伺服器：

```sh
python3 -m http.server 8377
# 開啟 http://localhost:8377
```

## 每天的冒險流程

1. 登入冒險者頁面（每日任務自動打勾）
2. 查詢伊藤忠商事（8001）今日股價，填入「本月冒險狀況」更新價格
3. 看 NPC 今日建議（村長／菇菇商人／肥肥商人）
4. 若有買入，記錄到「冒險日誌」→ LV UP！

## 核心機制

- **LV = 持股數**，100 股達成 LV MAX
- **EXP = 持股 ÷ 100**，以經驗值條呈現
- **稱號**：見習股東 → 穩健投資家 → 配息獵人 → 伊藤忠騎士 → 傳說投資家
- **成就徽章**：25 / 50 / 75 / 100 股，完成度越高徽章越亮，解鎖後顯示「解鎖済」
- **冒險規則**：自訂小買／中買／大買的跌幅門檻與上限股數（跌幅以現價相對平均成本計算），系統結合本月剩餘預算給出今日建議
- **平均成本**：採移動平均法，賣出以當時均價沖減
